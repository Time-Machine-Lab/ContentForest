## Why

营养汲取需要一个种子级工作入口，而不是分散在普通营养库页面和生长输入框里。用户需要在工作区中快速进入一个沉浸式营养工作台，集中查看营养卡片、与 Agent 研究和处理缺口建议。

## What Changes

- 在种子工作区增加类似基因库的“营养库 / 营养汲取”入口。
- 在种子专属营养库页面增加进入营养工作台的入口。
- 实现大尺寸悬浮工作台 Dialog，背景保留画布但弱化。
- 工作台采用三栏布局：左侧营养卡片、中间 Agent 工作区、右侧营养建议。
- 移动端或窄屏使用 Tab 切换三栏内容。

## Capabilities

### New Capabilities
- `nutrient-workbench-ui`: 定义营养工作台入口、悬浮 Dialog、三栏布局和响应式行为。

### Modified Capabilities

## Impact

- 影响种子工作区页面、种子专属营养库页面和导航/悬浮入口组件。
- 依赖后端提供种子级营养工作台聚合接口或现有营养、建议、会话接口组合。
- 不修改后端 API 或 SQL。
