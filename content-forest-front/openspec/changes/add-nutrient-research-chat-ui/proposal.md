## Why

营养工作台的核心是用户与 Agent 联网研究的持续交流。前端需要把普通对话和可沉淀营养块区分展示，让用户能将研究成果生成卡片或合并到已有卡片。

## What Changes

- 实现中间 Agent 研究工作区。
- 支持发送研究问题和查看实时/准实时响应。
- 区分展示普通沟通消息和可沉淀营养块。
- 支持基于可沉淀营养块创建未沉淀卡片或合并到当前卡片。
- 支持点击营养卡片后加载对应会话。

## Capabilities

### New Capabilities
- `nutrient-research-chat-ui`: 定义营养研究对话区、消息展示、可沉淀营养块和卡片会话加载交互。

### Modified Capabilities

## Impact

- 影响营养工作台中间区域组件。
- 依赖后端营养研究会话、消息提交和可沉淀营养块接口。
- 不修改 API 或 SQL。
