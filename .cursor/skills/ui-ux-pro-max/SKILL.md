# UI UX Pro Max

Use this skill when the user asks for UI/UX work such as building, designing, improving, reviewing, or fixing interfaces.

## What it provides
- UI styles and patterns
- Color palette guidance
- Typography pairings
- UX best practices and anti-patterns
- Stack-specific recommendations

## How to use it
1. Identify the product type, style direction, and target stack.
2. Apply the most suitable layout pattern and visual style.
3. Use accessible colors, readable typography, and responsive spacing.
4. Prefer SVG icons over emoji.
5. Include visible hover and focus states for interactive elements.
6. Respect reduced-motion preferences.
