## 新增需求

### 需求："我的生成器"页面

为已登录用户提供一个集中管理自己所有生成器（包括自建与已安装）的页面。

#### 场景：查看生成器列表

- **当** 用户访问「我的生成器」页面
- **则** 页面展示该用户创建的所有生成器，以及从市场安装的所有生成器
- **且** 两类生成器通过标签或分组进行区分
- **且** 每张卡片显示名称、描述、标签及最后更新时间

#### 场景：复制生成器调用指令

- **当** 用户点击某个生成器卡片上的「复制指令」按钮
- **则** 系统将该生成器的 Agent 调用指令（包含 generatorId）复制到剪贴板
- **且** 用户可将指令粘贴到 AI IDE 中，由 Agent 执行实际的内容生成
