## ADDED Requirements

### Requirement: 工作区聚合媒体资源摘要
系统 SHALL 在工作区快照中聚合果实媒体摘要和可引用媒体资源摘要。工作区快照 MUST 不返回媒体二进制内容。

#### Scenario: 果实节点包含媒体摘要
- **WHEN** 工作区快照包含挂载媒体的果实
- **THEN** 系统 MUST 返回该果实的媒体摘要
- **AND** 前端 MUST 能据此展示媒体数量或预览入口

#### Scenario: 工作区资源包含可引用媒体
- **WHEN** 当前种子存在可用于枝化生长引用的媒体资源
- **THEN** 系统 MUST 在工作区资源中返回媒体资源摘要
- **AND** 系统 MUST 不返回本机绝对路径或媒体二进制内容
