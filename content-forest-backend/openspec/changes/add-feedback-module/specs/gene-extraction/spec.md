## ADDED Requirements

### Requirement: 读取反馈证据上下文
系统 SHALL 支持基因汲取证据读取能力读取数据回流反馈上下文。当证据来源类型为 `feedback` 时，系统 MUST 能够基于反馈快照读取其发布记录、监控器语义、表现数据和用户观察，用作基因汲取 Agent 的上下文输入。

#### Scenario: 读取反馈快照作为证据
- **WHEN** 用户发起基因汲取任务并显式提供 `feedback` 类型证据来源
- **THEN** 系统 MUST 读取该反馈快照的表现数据和用户观察
- **AND** 系统 MUST 读取该反馈快照关联的发布记录上下文
- **AND** 系统 MUST 将这些内容作为基因汲取 Agent 的证据上下文

#### Scenario: 反馈证据不存在时拒绝汲取
- **WHEN** 用户发起基因汲取任务并提供不存在的反馈证据来源
- **THEN** 系统 MUST 拒绝创建或执行基因汲取任务
- **AND** 系统 MUST 返回反馈证据不存在的错误

#### Scenario: 创建反馈快照不自动触发汲取
- **WHEN** 用户创建或编辑反馈快照
- **THEN** 系统 MUST NOT 自动创建基因汲取任务
- **AND** 系统 MUST NOT 自动创建正式基因经验
