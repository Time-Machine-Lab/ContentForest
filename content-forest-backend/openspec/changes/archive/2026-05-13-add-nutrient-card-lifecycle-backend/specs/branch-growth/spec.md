## ADDED Requirements

### Requirement: 支持未沉淀营养临时引用
枝化生长 SHALL 支持在一次生长请求中引用当前种子下的未沉淀营养卡片。未沉淀营养卡片只作为本次生长临时资料，不因此成为正式营养内容。

#### Scenario: 临时引用未沉淀营养
- **WHEN** 用户发起枝化生长并提供当前种子下的未沉淀营养卡片引用
- **THEN** 系统 MUST 校验该卡片归属于当前种子且未归档
- **AND** 系统 MUST 将该卡片作为临时营养资料加入 Agent 授权范围
- **AND** 系统 MUST 在 Agent 输入中标记该资料为未沉淀候选

#### Scenario: 拒绝其他种子的未沉淀营养
- **WHEN** 用户发起枝化生长并引用其他种子的未沉淀营养卡片
- **THEN** 系统 MUST 拒绝创建生长任务
- **AND** 系统 MUST 返回引用越权错误

#### Scenario: 拒绝归档卡片引用
- **WHEN** 用户发起枝化生长并引用已归档营养卡片
- **THEN** 系统 MUST 拒绝创建生长任务
- **AND** 系统 MUST 返回营养卡片不可引用错误
