## ADDED Requirements

### Requirement: 启动时初始化运行时内容目录
系统 SHALL 在后端应用启动装配阶段初始化运行时内容根目录，并创建第一期所需的标准内容子目录。

#### Scenario: 首次启动时创建目录骨架
- **WHEN** 后端应用在本地环境首次启动
- **THEN** 系统 MUST 创建运行时内容根目录
- **AND** 系统 MUST 创建种子、果实、营养、生成器、附件和临时文件对应的标准子目录

#### Scenario: 目录已存在时重复启动
- **WHEN** 后端应用启动且运行时内容目录已经存在
- **THEN** 系统 MUST 保持启动过程幂等
- **AND** 系统 MUST 不清空或覆盖已有内容文件

### Requirement: 内容根目录由应用配置提供
系统 SHALL 通过应用配置提供运行时内容根目录，业务模块 MUST 不直接读取环境变量或拼接本机绝对路径。

#### Scenario: 使用默认内容根目录启动
- **WHEN** 未提供自定义内容根目录配置
- **THEN** 系统 MUST 使用本地开发默认内容根目录

#### Scenario: 使用自定义内容根目录启动
- **WHEN** 提供自定义内容根目录配置
- **THEN** 系统 MUST 使用该配置初始化内容根目录
- **AND** 内容访问适配器 MUST 基于该根目录读写内容本体

### Requirement: 内容位置统一保存相对路径
系统 SHALL 在数据库系统事实中仅保存内容本体的相对位置，MUST NOT 保存本机绝对路径。

#### Scenario: 保存种子 Markdown 内容位置
- **WHEN** 系统保存种子 Markdown 内容本体
- **THEN** 系统 MUST 在系统事实中保存相对内容位置
- **AND** 系统 MUST NOT 保存本机绝对路径

#### Scenario: 内容访问层解析内容位置
- **WHEN** 内容访问适配器读取或写入内容本体
- **THEN** 系统 MUST 将相对内容位置解析到运行时内容根目录下

### Requirement: 内容访问适配器阻止路径越界
系统 SHALL 阻止内容访问适配器访问运行时内容根目录之外的路径。

#### Scenario: 拒绝越界相对路径
- **WHEN** 内容访问适配器收到包含越界片段的内容位置
- **THEN** 系统 MUST 拒绝该路径
- **AND** 系统 MUST 不读取或写入运行时内容根目录之外的文件

#### Scenario: 拒绝绝对内容位置
- **WHEN** 内容访问适配器收到本机绝对路径作为内容位置
- **THEN** 系统 MUST 拒绝该路径
- **AND** 系统 MUST 不将该路径作为内容本体位置保存

### Requirement: 文档同步运行时文件系统规则
系统 SHALL 将运行时文件系统根目录、标准目录骨架、相对路径规则和 Bootstrap 职责同步到项目架构与后端开发规范文档。

#### Scenario: 架构文档包含运行时文件系统设计
- **WHEN** 开发者查看架构设计文档
- **THEN** 文档 MUST 描述运行时内容根目录和标准内容目录骨架

#### Scenario: 后端规范包含路径边界规则
- **WHEN** 开发者查看后端开发规范文档
- **THEN** 文档 MUST 说明目录初始化由 Bootstrap 负责
- **AND** 文档 MUST 说明数据库内容位置只能保存相对路径
